# Font Files

To complete the font setup, please add the following font files to this directory:

## Required Files:

### Momo Trust Display (Logo only)
- `MomoTrustDisplay-Regular.woff2` and `MomoTrustDisplay-Regular.woff` (weight: 400)

### Gellix (Main site font)
- `Gellix-Regular.woff2` and `Gellix-Regular.woff` (weight: 400)
- `Gellix-Medium.woff2` and `Gellix-Medium.woff` (weight: 500)
- `Gellix-SemiBold.woff2` and `Gellix-SemiBold.woff` (weight: 600)
- `Gellix-Bold.woff2` and `Gellix-Bold.woff` (weight: 700)

## Where to get fonts:

**Momo Trust Display:**
- Used exclusively for the "MGH Tech" logo text in the navbar

**Gellix:**
- Commercial font available from:
  - MyFonts: https://www.myfonts.com/collections/gellix-font-lettertype
  - Fontspring: https://www.fontspring.com/fonts/type-dynamic/gellix

## Fallbacks:

If fonts are not available, the site will gracefully fall back to:
- Logo: Gellix → Arial → sans-serif
- Body: Arial → sans-serif

## Format Conversion:

If you have TTF or OTF files, you can convert them to WOFF/WOFF2 using:
- https://cloudconvert.com/woff-converter
- https://transfonter.org/
